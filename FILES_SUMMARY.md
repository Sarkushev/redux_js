## 🔧 API ФУНКЦИИ (1 файл)

### ✅ `src/api/api.js`
**Назначение:** API функции с имитацией задержки  
**Функции:**
- `fetchCars()` - Все машины (1500ms)
- `fetchCarById(id)` - Одна машина (1000ms)
- `fetchArticles()` - Все статьи (2000ms)
- `fetchArticleById(id)` - Одна статья (1200ms)
- `fetchProducts()` - Все товары (1800ms)
- `fetchProductById(id)` - Один товар (1000ms)
- `fetchCategories()` - Все категории (1300ms)
- `fetchCategoryById(id)` - Одна категория (800ms)
- `simulateDelay(ms)` - Вспомогательная функция
