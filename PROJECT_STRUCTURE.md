## 🚀 Как всё работает

### 🔄 DATA FLOW (Поток данных):

```
1. КОМПОНЕНТ МОНТИРУЕТСЯ
        ↓
2. useEffect() → dispatch(loadData)
        ↓
3. Redux Thunk активируется
        ↓
4. API функция вызывается
        ↓
5. simulateDelay(ms) имитирует запрос (1-2 сек)
        ↓
6. JSON данные возвращаются
        ↓
7. Redux обновляет state
        ↓
8. Компонент перерендеривается
        ↓
9. Данные отображаются на экране
```

### 📱 ГЛАВНАЯ СТРАНИЦА:


## 🔧 Как расширить

### Добавить новый контент:

1. **Создать JSON файл:**
   ```json
   // src/mock-data/mydata.json
   [
     { id: 1, name: "...", ... },
     { id: 2, name: "...", ... }
   ]
   ```

2. **Создать API функции:**
   ```javascript
   // src/api/api.js
   export const fetchMyData = async () => {
     await simulateDelay(1500);
     return myData;
   };
   ```

3. **Создать Redux слайс:**
   ```javascript
   // src/store/mySlice.js
   export const loadMyData = createAsyncThunk(
     'my/loadMyData',
     async () => {
       return await fetchMyData();
     }
   );
   ```

4. **Создать компонент:**
   ```jsx
   // src/components/MyList.jsx
   export function MyList() {
     // Используйте Redux selectors и dispatch
   }
   ```

5. **Добавить в App:**
   ```jsx
   // src/App.jsx
   <MyList />
   ```
