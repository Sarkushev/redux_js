```javascript
{
  user: {
    users: [],
    currentUser: null,
    loading: false,

  },
  articles: {
    list: [],
    selectedArticle: null,
    loading: false,
    error: null,
  },
  products: {
    list: [],
    selectedProduct: null,
    loading: false,
    error: null,
  },
  categories: {
    list: [],
    loading: false,
    error: null,
  }
}
```

### API Функции:
- `fetchCars()` - Все машины с задержкой 1.5s
- `fetchCarById(id)` - Одна машина с задержкой 1s
- `fetchArticles()` - Все статьи с задержкой 2s
- `fetchArticleById(id)` - Одна статья с задержкой 1.2s
- `fetchProducts()` - Все товары с задержкой 1.8s
- `fetchProductById(id)` - Один товар с задержкой 1s
- `fetchCategories()` - Все категории с задержкой 1.3s
- `fetchCategoryById(id)` - Одна категория с задержкой 0.8s

### Redux Thunks:
- `loadCars` / `loadCarById`
- `loadArticles` / `loadArticleById`
- `loadProducts` / `loadProductById`
- `loadCategories` / `loadCategoryById`

## 🚀 ПРОЦЕСС РАБОТЫ

### Загрузка данных:
1. Компонент монтируется
2. useEffect вызывает dispatch()
3. createAsyncThunk активируется (pending)
4. API функция вызывается
5. simulateDelay имитирует задержку (1-2 сек)
6. JSON данные возвращаются
7. Redux переходит в fulfilled
8. Компонент перерендеривается
9. Данные демонстрируются

### LIST/DETAIL для статей:
1. Изначально показывается список статей (5 карточек)
2. Кликните на карточку
3. Состояние меняется (selectedId = article.id)
4. Компонент показывает полное содержание статьи
5. Нажмите "Вернуться к списку"
6. Состояние сбрасывается обратно в список
