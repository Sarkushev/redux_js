# ⚡ Быстрый старт

## Что было реализовано:

✅ **JSON данные** - Все данные загружаются из JSON файлов:
   - `src/mock-data/cars.json` - Автомобили
   - `src/mock-data/articles.json` - Статьи (с LIST/DETAIL)
   - `src/mock-data/products.json` - Товары
   - `src/mock-data/categories.json` - Категории

✅ **Асинхронная загрузка с задержкой** - В `src/api/api.js`:
   ```javascript
   const simulateDelay = (ms = 1500) => {
     return new Promise(resolve => setTimeout(resolve, ms));
   };
   ```
   - Автомобили: 1.5 сек
   - Статьи: 2 сек
   - Товары: 1.8 сек
   - Категории: 1.3 сек

✅ **Несколько контентов на главной**:
   1. 🚗 Автомобили - LIST/DETAIL функциональность
   2. 📰 Статьи - LIST/DETAIL функциональность (кликайте на статью)
   3. 🛒 Товары - Каталог с фильтрацией
   4. 📂 Категории - Навигация по категориям

✅ **LIST/DETAIL реализована для Статей** (`ArticleList.jsx`):
   - Нажмите на любую статью для просмотра полного содержания
   - Кнопка "Вернуться к списку" для возврата

## 🚀 Запуск проекта

```bash
# 1. Перейти в папку проекта
cd react-redux-site

# 2. Установить зависимости (если ещё не установлены)
npm install

# 3. Запустить dev сервер
npm run dev

# 4. Открыть браузер на http://localhost:5173
```

## 📝 Как модифицировать

### Изменить время загрузки
В файле `src/api/api.js` измените время в миллисекундах:
```javascript
export const fetchCars = async () => {
  await simulateDelay(3000); // Теперь 3 секунды
  return carsData;
};
```

### Добавить новые данные
1. Отредактируйте JSON файлы в `src/mock-data/`
2. Данные автоматически обновятся

### Добавить новый контент
1. Создайте новый JSON файл в `src/mock-data/`
2. Создайте API функцию в `src/api/api.js`
3. Создайте Redux слайс в `src/store/`
4. Создайте компонент в `src/components/`
5. Добавьте импорт и используйте в `App.jsx`
